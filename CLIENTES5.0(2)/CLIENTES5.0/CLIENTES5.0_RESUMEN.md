# CLIENTES5.0 — Kargax

Fecha: 2026-05-25

## Tesis de venta

Kargax debe venderse como una capa de control operativo para empresas con rutas B2B, bodega y entregas recurrentes. El mensaje central no es “software logístico”, sino: **menos reclamos, menos evidencia dispersa y cada entrega cerrada con PIN/foto/novedad/reporte**.

## ICP

- Fabricantes o distribuidores B2B con bodega y rutas.
- Operadores con flota propia o tercerizada.
- Empresas con productos frescos, frágiles, pesados, voluminosos o con muchas referencias.
- Decisores: COO, Director de Operaciones, Jefe de Logística, Supply Chain Manager, Jefe de Despachos, Gerente Comercial B2B.

## Criterios de scoring

- Dolor que Kargax resuelve: 0-20.
- Urgencia: 0-15.
- Capacidad de pago: 0-15.
- Facilidad de contacto: 0-10.
- Probabilidad de compra: 0-15.
- Uso recurrente: 0-15.
- Encaje estratégico: 0-10.
