# 09 — Copy seguro de producto

## Principio de copy

KargaX no debe sonar como banco. Debe sonar como ledger operativo logístico.

Evitar:

```text
cuenta bancaria
custodiamos tu dinero
saldo garantizado
rendimientos
depósito
nómina liberada a billetera
```

Usar:

```text
saldo marketplace retirable
ledger operativo
liquidación privada externa
comprobante de pago
pago confirmado por KargaX
pago realizado por tu empresa
```

## Marketplace

### Tarjeta principal

```text
Marketplace — saldo retirable
Dinero de rutas marketplace pagadas y completadas con evidencia. Puedes solicitar retiro cuando cumpla los controles de KargaX.
```

### Payout automático

```text
Pago automático en proceso
Tu retiro fue creado y entró a la cola de pago. KargaX lo concilia antes de marcarlo como pagado.
```

### Sin método de retiro

```text
Agrega un método de retiro
Tu saldo marketplace está disponible, pero necesitamos un destino válido para procesar el pago.
```

## Flota privada

### Sección

```text
Liquidaciones privadas
Tu empresa registra aquí pagos y comprobantes. Estos pagos se realizan por fuera de KargaX y no aumentan tu saldo retirable.
```

### Pendiente externo

```text
Pendiente de pago externo
Tu empresa aún no ha marcado esta liquidación como pagada ni ha cargado comprobante.
```

### Comprobante cargado

```text
Comprobante cargado
Tu empresa cargó soporte del pago. KargaX conserva el registro para trazabilidad.
```

### Pagado externo

```text
Pagado por tu empresa
Este pago fue marcado como realizado por fuera de KargaX. Revisa el comprobante si necesitas soporte.
```

## Retiro bloqueado por saldo no elegible

```text
Solo puedes retirar saldo marketplace confirmado. Las liquidaciones privadas se pagan por fuera con comprobante y no hacen parte de tu saldo retirable.
```

## Admin empresa

```text
Registra liquidaciones y comprobantes de tu flota privada sin cambiar tu forma actual de pago. KargaX organiza el historial, estados y soporte por conductor.
```

## Admin KargaX

```text
Payout en revisión manual
El intento requiere validación operativa antes de marcarlo como pagado o reintentarlo.
```
