# CERRADO - Sprint 31: Home Comercial Flota Privada + Despacho + Evidencia

## Objetivo

Reposicionar el home para vender KargaX como sistema operativo de logistica diaria: flota privada, despacho, evidencia, wallet y reportes.

## Implementado

- Home `/` refactorizado con mensaje principal:
  - flota privada.
  - despacho desde bodega.
  - evidencia de cargue/entrega.
  - pagos/wallet.
  - reporte mensual.
- CTAs:
  - `Agendar piloto`.
  - `Ver como funciona`.
  - login/crear cuenta.
  - publicar oferta/explorar cargas.
- Secciones:
  - flujo bodega -> despacho -> viaje -> evidencia -> wallet -> reporte.
  - problemas que resuelve.
  - modulos principales.
  - piloto 7 dias.
- Se elimino lenguaje de promesa financiera activa:
  - no adelantos.
  - no credito.
  - no pago en dos horas.
- Pagina `/para-camioneros` para adquisicion secundaria.

## Posicionamiento

KargaX se vende primero a empresas que tienen friccion real en:

- control de conductores privados.
- evidencia operacional.
- despachos conectados a bodega.
- pagos y retiros trazables.
- reportes para gerencia/contabilidad.

## QA

- Home responsive mobile/desktop.
- CTAs correctos.
- Sin copy de credito/adelantos/pago expres.
- Landing camioneros responsive.
- Link WhatsApp de referido funciona.
