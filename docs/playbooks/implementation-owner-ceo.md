# Implementation Playbook: Owner/CEO

Goal: show KargaX as the production command layer for marketplace, warehouse OS, wallet, lending and holding governance.

1. Start from the public homepage and position KargaX as an enterprise operating system.
2. Open the company workspace and implementation route.
3. Show first warehouse, first trip, settlement timeline and wallet liquidity.
4. Show a live advance in portfolio with policy and exposure snapshots.
5. Close in `/corporativo` with SLA queue, treasury and multi-company control.

Length target: 15 minutes.
